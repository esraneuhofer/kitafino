Um Ihnen den Einstieg zu erleichtern, haben wir einige Schritte für Sie vorbereitet:

Unterstützung: Auf allen Seiten können Sie Hilfe erhalten, indem Sie auf das blaue Fragezeichen in der rechten unteren Ecke klicken.

Hilfethemen: Im Menüpunkt "Hilfe" finden Sie verschiedene Hilfethemen mit detaillierten PDFs.

Videoanleitungen: Besuchen Sie unseren YouTube-Kanal (https://youtube.com), um Videoanleitungen zu den einzelnen Punkten von Cateringexpert anzusehen.

Navigation

Um zwischen den einzelnen Menüpunkten zu wechseln, klicken Sie auf das Menüsymbol: [Menu ] in der oberen rechten Ecke.

Anschließend öffnet sich das Menü, in dem Sie die gewünschte Seite auswählen können.

Ersten Schritte

Anmeldung: Als erstes müssen Sie einen Verpflegungsteilnehmer (Kind) anmelden. Klicken Sie hierfür auf den Menüpunkt "Verpflegungsteilnehmer" und folgen Sie den Anweisungen.

Konto aufladen: Um eine Bestellung aufgeben zu können, müssen Sie Ihr Konto aufladen. Klicken Sie hierfür auf [Kontoübersicht] und folgen Sie den Anweisungen.

Bestellen: Nachdem Sie Ihr Konto aufgeladen haben, können Sie Bestellungen aufgeben. Sie haben die Möglichkeit, die Bestellung täglich aufzugeben oder eine Dauerbestellung einzutragen. Gehen Sie hierfür entweder auf den Menüpunkt [Bestellungen] oder [Dauerbestellungen].

Nähere Informationen finden Sie auf der jeweiligen Seite oder unter dem Menüpunkt [Hilfe].

Wir wünschen Ihnen viel Spaß mit der Anwendung.
